Welcome to StarMade

        
StarMade is a 3D voxel sandbox space shooter where players can
            explore the universe, create and fly space ships, build space
            stations, mine and craft resources, colonize planets, form
            alliances, fight for territories, and more.

        
In this procedurally-generated universe of StarMade, you're only
            limited by your imagination. Play on your own or with friends and
            journey into the new frontier.

Changelog

            
v3.0.0

            

                

                    
                    
                        
Major Update

                        
    Conveted all CoffeScript code to JavaScript for easier maintainabiltiy.

                        
    Converted all Jade code to HTML for easier maintainability.

                        
    The Launcher now supports automatic Java version switching from Java 8 to Java 18.

                        
    Game versions prior to the Universe Update (< 0.3.000) will now automatically switch to Java 8 instead of Java 7 for increased performance.

                        
    Game versions after the Universe Update (>= 0.3.000) will now automatically switch to Java 18.

                    		
                

            

            
v2.1.3

            

                

                    
                    The launcher now uses javaw.exe to launch the game on Windows. This should help mitigate issues with the recent buggy nVidia graphics driver. (version
                        378.49, released on January 24th, 2017)
                    
                

            

            
v2.1.2

            

                

                    
                    Fixed displaying of Citizen Broadcasts. This will allow us to display important messages and notifications. Note that these messages are only displayed once, so be sure to read
                        them.
                    
                

            

            
v2.1.1

            

                

                    
                    Added changelog
                

                

                    
                    Added backup dialog, which optionally creates a compressed copy of the game configs, worlds, and blueprints prior to updating. Support for both .zip
                        and .tar.gz formats.
                    
                

                

                    
                    Launcher now reads the game version from its version.txt; this allows the launcher to detect Steam updates, and also (unofficially) supports
                        concurrent game versions. We will add official support in a later update.
                    
                

                

                    
                    Added custom java options field to the launcher options dialog.
                

                

                    
                    Added validation to memory settings window
                

                

                    
                    Added validation to guest logins
                

                

                    
                    Fixed selecting custom Java paths
                

                

                    
                    Fixed news post scrubbing. There should no longer be missing sections.
                

                

                    
                    Improved --help and added version information.
                

                

                    
                    Numerous backend improvements, including a rewrite of memory settings handling and updates to logging.
