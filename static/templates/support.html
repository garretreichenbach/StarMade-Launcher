For OSX (especially yosemite), you need to install the Java Development Kit (JDK) to have the system register the Java Runtime Environment (JRE). There are various reasons for lower game performance. The most common reasons are:
    1. Outdated video drivers. Please look for updated drivers on your video card manufacturer's
      (and your computer manufacturer's) websites.
    2. Multi-GPU systems, like notebooks, tend to run java with the powersaving GPU.
      Right click the launcher executable and use the option: "Execute with: -> High performance GPU"
    3. Try increasing the amount of RAM available to the game.
      You can find the memory settings button to the left of the [LAUNCH] button below.
    4. Try decreasing the "Max View Segments" option within the game's settings.
      This has a small effect on the game's graphics, but can have a large impact on framerate.
